#🎓 Student Management System </br>
#A robust, modular Command Line Interface (CLI) application built with Python and MySQL. This system allows users to seamlessly </br> perform #CRUD (Create, Read, Update, Delete) operations on student records, demonstrating core database integration and software architecture </br>#principles.</br>

#✨ Features </br>
#Add Students: Register new students with their name, email, and enrolled course.</br>

#View Records: Display a formatted table of all students currently in the system.</br>

#Update Courses: Modify a student's enrolled course using their unique Student ID.</br>

#Delete Records: Remove students from the database securely.</br>

#Data Persistence: All information is stored permanently using a MySQL backend. </br>

#🛠️ Technologies Used </br>
#Python 3.x: Core application logic and user interface. </br>

#MySQL: Relational database management system. </br>

#mysql-connector-python: Driver to bridge the Python application and MySQL server. </br>
